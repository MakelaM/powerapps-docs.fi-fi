# <a name="discover-apps-via-appsource"></a>Löydä sovelluksia AppSourcen kautta
PowerAppsilla laatimasi ja jakamasi sovellukset ovat löydettävissä ja niitä käytetään Dynamics 365:ssä Internetissä. Siellä näkyvät kaikki sovelluksesi, sekä ne, jotka olet varta vasten valinnut käyttöön (käynnistämällä jakosähköpostiviestin kautta tai avaamalla AppSourcesta), että ne, jotka järjestelmänvalvoja on hankkinut käyttöösi. Näihin sisältyvät kaikki sovelluksesi PowerAppsista sekä myös Dynamics 365 -sovelluksia Microsoftilta. Voit etsiä sovelluksia, suodattaa ympäristön perusteella ja kiinnittää useimmin käyttämäsi sovellukset sivun yläreunaan.

  ![Sovellukset Dynamics 365:ssä](./media/app-source/apps-dynamics365.png)

## <a name="find-apps-via-the-appsource-organization-gallery"></a>Etsi sovelluksia AppSource organisaatiovalikoiman kautta
Microsoft AppSource on nyt upotettu [Dynamics 365](http://home.dynamics.com):n aloitussivulle ja kaikkiin yleisiin siirtymistoimenpiteisiin. Se sisältää nyt yksityisen sovellusvalikoiman, joka on yrityksessäsi käytettävissäsi. Valitse **Hanki lisää sovelluksia** aloitussivulta tai tehtäväruudusta, ja siirry **oma organisaatio** välilehteen nähdäksesi sovellukset, jotka ovat käytettävissäsi.

![Sovellukset Dynamics 365:ssä](./media/app-source/getmoreapps.png)

PowerAppsissa laaditut ja käyttöoikeusryhmän tai koko yrityksen kesken **Käyttäjä**-käyttöoikeuksilla jaetut sovellukset tulevat näkyviin tähän sen sijaan, että ne sotkisivat oletusarvoisesti aloitussivusi. Kun saat sovelluksen AppSourcesta, se tulee silloin näkyviin aloitussivulle ja on helposti kaikkialta käytettävissä.

  ![Sovellukset Dynamics 365:ssä](./media/app-source/appsource.png)
