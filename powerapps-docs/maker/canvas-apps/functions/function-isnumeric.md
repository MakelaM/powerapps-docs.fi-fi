# <a name="isnumeric-function-in-powerapps"></a>IsNumeric-funktio PowerAppsissa
Testaa, onko arvo numeerinen.

## <a name="description"></a>Kuvaus
**IsNumeric**-funktio testaa, onko arvo numeerinen.  Muunlaisia arvoja ovat esimerkiksi Boolean, merkkijono, [taulukko](../working-with-tables.md) ja [tietue](../working-with-tables.md#records).

Palautusarvo on totuusarvo **tosi** tai **epätosi**.

## <a name="syntax"></a>Syntaksi
**IsNumeric**( *arvo* )

* *Arvo* – pakollinen. Testattava arvo.
