# <a name="publish-code-components-on-appsource"></a>Koodi komponenttien julkaiseminen Appsourcessa

Kun olet testannut koodi komponentit ja haluat sen jaettavaksi muille, jotta voit käyttää sitä, sinun on julkaistava se Microsoft AppSource.

Tarkempia tietoja siitä, miten koodi komponentit julkaistaan Appsourcessa, on kohdassa [sovelluksen julkaiseminen appsourcessa](/powerapps/developer/common-data-service/publish-app-appsource).
