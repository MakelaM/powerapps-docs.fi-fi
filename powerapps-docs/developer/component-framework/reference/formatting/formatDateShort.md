# <a name="formatdateshort"></a>formatDateShort

[!INCLUDE [formatdateshort-description](includes/formatdateshort-description.md)]

## <a name="syntax"></a>Syntaksi

`context.formatting.formatDateShort(value, includeTime);`

## <a name="available-for"></a>Käytettävissä kohteelle 

Mallipohjaiset sovellukset ja pohjaan perustuvat sovellukset (kokeellinen esikatselu)

## <a name="parameters"></a>Parametrit

| Parametrin nimi|Tyyppi|Pakollinen|Kuvaus|
| ------------- |----|--------|-----------|
|arvo|`Date`|Kyllä|Muotoilun arvo päivämäärä.|
|includeTime|`boolean`|Kyllä|Näytetäänkö aika muotoiltuna arvona.|

## <a name="return-value"></a>Palautusarvo

Tyyppi: `string`
